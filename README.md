# HireHub (Frontend)

HireHub is a web-based application that connects skilled workers such as barbers, plumbers, teachers, and technicians with people who need their services in Nigeria.

## Features
- Browse skilled workers by category
- View worker profiles
- Search and filter services
- Responsive user interface

## Built With
- React.js
- JavaScript
- HTML
- CSS
- Bootstrap
- Tailwind
